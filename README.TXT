Simple Website Structure Diagram
/ (Root Directory)
├── bin
│   └── www (Script to start the server)
├── public (Static assets directory)
│   ├── images (Stores images)
│   ├── javascripts (Stores JavaScript files)
│   └── stylesheets (Stores CSS files)
│       └── style.css
├── routes (Routes)
│   ├── index.js (Root route handler)
│   ├── dashboard.js (Dashboard route handler)
│   └── user.js (User route handler)
├── views (View templates)
│   ├── error.ejs (Error page template)
│   ├── index.ejs (Home page template)
│   ├── signin.ejs (Sign in page template)
│   ├── signup.ejs (Sign up page template)
│   ├── dashboard.ejs (Dashboard page template)
│   ├── profile.ejs (Profile page template)
│   └── settings.ejs (Settings page template)
├── app.js (Main application file)
├── package.json (Application dependencies and settings)
└── package-lock.json (Ensures consistent dependency versions)
